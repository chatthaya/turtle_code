## How to Run the Program
1. Install Python 3.x on your computer.
2. Run the program using the following command:

   python art_generator.py

3. When the menu appears, enter a number from 1 to 9 to generate the corresponding art piece.
4. A Turtle Graphics window will open and display your artwork.
